# 🚀 Project Overview

Omnia Mart is a modern full-stack e-commerce platform designed with a professional architecture using React, Express, Node.js and MySQL.

The project includes:

- Customer Shopping Portal
- Admin Dashboard
- Product Management
- Authentication System
- Image Upload
- Shopping Cart
- REST API
- MVC Backend
